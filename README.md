Installation

Create a virtual environment before running the program. You can install it using one of two methods:

## Method 1: From GitHub
1. Download the code  
2. Create and activate venv using `uv sync`  
3. Add `.env` file  
4. Run:  
```bash
uv run python -m app.main
```

## Method 2: From PyPI
```bash
pip install smartfeed-ai
```
2. create and configure a .env file
```bash
smartfeed
```
